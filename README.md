# Simple Todo App

This project consists of refactoring existing HTML/JS to update the directory structure using JS modules.

## Functionality

Specificially, it allows users to toggle checkboxes to mark todo items, add new todo items, and delete todo items. In addition, users are met with error messages if fields are not filled correctly, error messages reset after form submission, and the date appears next to the todo item.

## Technology

- JavaScript ES6+ - Classes, arrow functions, template literals, destructuring
- HTML5 - Semantic markup, form validation attributes
- CSS3 - Modern styling, BEM methodology for class naming
- DOM Manipulation - Event listeners, element selection, dynamic content

JavaScript Concepts We've Covered:

- Object-Oriented Programming - Classes (FormValidator, Todo)
- ES6 Modules - Import/export statements
- Array Methods - forEach, some, Array.from
- Event Handling - addEventListener, form submissions
- Form Validation - Built-in HTML5 validation API
- Template Elements - HTML templates and cloning
- External Libraries - UUID for generating unique IDs

Project Structure:

- Modular Architecture - Separate files for different functionalities
- Component-based Design - Reusable Todo and FormValidator classes
- Configuration Objects - Centralized settings in constants.js
